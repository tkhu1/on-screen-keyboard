# on-screen-keyboard
Custom on-screen keyboard GUI in C++ developed via the Qt framework suite.
The keyboard keys are arranged in eight rows, with each row containing ten keys.

Layout is like so:
-----------------------------
|  1 2 3 4 5 6 ~ ` ! @      |
|  7 8 9 0 - _ = + # $      |
|  a e i o u y % ^ & *      |
|  q w r t p s ( ) [ ]      |
|  d f g h j k { } \ |      |
|  l z x c v b ; : ' "      |
|  n m T B O E , . < > / ?  |
|  SPACE      ENTER         |
-----------------------------

Note that T = Tab
          B = Backspace
          O = Ok
          E = Esc
